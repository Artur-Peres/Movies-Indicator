# Movies-Indicator

## Descrição:
  Movies-Indicator é um programa desenvolvido em Java que utiliza uma árvore de decisão para indicar filmes de acordo com as preferências do usuário.
  O sistema apresenta perguntas sobre gêneros e características dos filmes e, com base nas respostas fornecidas, retorna uma lista de pelo menos três filmes que correspondem ao perfil desejado.

  O programa é totalmente executado no console e foi criado como parte da disciplina de Matemática Aplicada à Computação, demonstrando o uso de lógica de decisão e estrutura de dados aplicadas a um caso prático.

## Funcionalidades:

  Exibe um menu inicial com opções de gêneros de filmes (como ação, aventura, comédia, drama, terror, entre outros).
  
  Seleciona uma árvore de decisão específica para o gênero escolhido.
  
  Realiza perguntas que guiam o usuário até uma recomendação final.
  
  Retorna uma lista com pelo menos três filmes que atendem aos critérios informados.

## Como Executar o Projeto:

  Pré-requisitos:

  Ter o Java JDK 17 ou superior instalado.
  
  Ter um terminal configurado com as variáveis de ambiente do Java.
  
  Passos:
  
  Clonar o repositório:
  git clone https://github.com/seuusuario/Movies-Indicator.git
  
  Acessar o diretório do projeto:
  cd Movies-Indicator/src

  Compilar os arquivos Java:
  javac Main.java

  Executar o programa:
  java Main

Após isso, o programa será iniciado no console, exibindo o menu de gêneros e as perguntas para o usuário escolher.

## Objetivo do Projeto:
  Este projeto foi desenvolvido como parte de um estudo sobre a estrutura de dados "árvores", aplicando o conceito em uma situação prática: a recomendação de filmes baseada em decisões tomadas pelo usuário.
